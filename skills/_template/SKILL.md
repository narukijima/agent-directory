# `<skill-name>` — <一行説明>

## 発動条件

- 利用者が「`<skill-name>`を使って」と明示したとき
- <このスキルを選ぶべき依頼文の具体例を書く>

## 目的

依頼された対象を、定義された手順で分析し、契約どおりの形式で結果を返す。

## 手順

1. 依頼から分析対象と前提条件を確認する。不明なら着手前に利用者へ確認する。
2. 根拠が必要なら `knowledge/wiki/index.md` から該当ページ・原資料を参照する。
3. 中間ファイルは `.tmp/` に置いて分析する。
4. 結果を「出力契約」の形式でまとめ、`.tmp/` を片付ける。

## 出力契約

```text
## 結果
- 判定: <判定値>
- 根拠: <参照したKnowledgeページ・原資料へのリンク>
- 限界: <この分析で分からないこと>
```

## 禁止事項

- この `SKILL.md` を読まずに実行しない。
- 根拠のない判定を出力しない。参照元を必ず示す。
- 秘密情報を出力・保存しない。
- `.tmp/` 以外に中間ファイルを残さない。
